```mermaid
flowchart TD
    A[Client Request /api/get-stocks] --> B[Server Handler];
    B --> C[Create sampleStockData array];
    C --> D[Respond with JSON {success:true, data:sampleStockData}];
    D --> E[Client receives data];
```

**Endpoint**

`GET /api/get-stocks`

**Response**

