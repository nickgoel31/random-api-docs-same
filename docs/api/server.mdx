# Overview

This file `server.js` implements a simple Express server that provides two API endpoints for retrieving software feature data.

## Endpoints

### GET `/api/feature/random`

Returns a random feature object.

Response format:
```json
{
  "success": true,
  "timestamp": "2023-01-01T12:00:00.000Z",
  "data": {
    "id": 1,
    "name": "Dark Mode Toggle",
    "difficulty": "Easy",
    "category": "UI"
  }
}
```

### GET `/api/features`

Returns the full list of features.

Response format:
```json
{
  "success": true,
  "count": 6,
  "data": [
    {
      "id": 1,
      "name": "Dark Mode Toggle",
      "difficulty": "Easy",
      "category": "UI"
    }
    // ... other features
  ]
}
```

The server listens on port `3001`. Start it with `node server.js` and access the endpoints at `http://localhost:3001`.
